# Clarificaciones

Decidimos que, dado que `src/*.md` contienen mucha información (verificados por tamaño, algunos con varios KB), lo mejor es enlazarlos desde el `README.md` principal en vez de incrustar todo su contenido, que lo haría ilegible e inmanejable. El alcance de la tarea "Unificar readme.md" se entiende como "Centralizar los puntos de acceso a la documentación desde el README.md principal".
