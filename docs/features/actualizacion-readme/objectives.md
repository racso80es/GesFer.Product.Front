# Objetivo

Unificar el archivo `README.md` principal con la documentación y guías ubicadas en `src/*.md` y adecuar el contenido para que sea un reflejo de este, garantizando que todo esté referenciado correctamente sin saturar el documento principal.

# Alcance

Modificación del archivo `README.md` en la raíz del proyecto para enlazar y detallar el propósito de cada archivo Markdown presente en `src/`. No se eliminarán los archivos originales en `src/` ya que contienen detalles técnicos valiosos y extensos que sobrecargarían el README raíz.

# Ley Aplicada

Single Source of Truth (SSOT).
