# Especificación
El archivo `README.md` de la raíz agrupará la información contenida en:
- `src/config/README.md` (Configuración de Entornos y Variables).
- `src/tests/README.md` (Tests de Playwright).
Se eliminarán los archivos originales en `src/config/` y `src/tests/`.
