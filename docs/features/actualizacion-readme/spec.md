# Especificación
Unificar el contenido del `README.md` de la raíz con cualquier posible archivo `README.md` localizado en `src/`. El documento unificado residirá en la raíz del repositorio, y cualquier archivo `README.md` en `src/` será eliminado o fusionado para evitar redundancia y centralizar la documentación como se especifica en la memoria del sistema.
