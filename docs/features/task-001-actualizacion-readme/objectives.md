# Objetivos

El objetivo de esta tarea es consolidar y unificar la documentación de `README.md` principal del proyecto y varios archivos de ayuda/instrucciones sueltos en el directorio `src/`. Esto servirá para mantener el proyecto más organizado y proveer una "Single Source of Truth" de instrucciones principales en la raíz del repositorio, moviendo guías especializadas a directorios `docs/architecture` y `docs/testing`.
