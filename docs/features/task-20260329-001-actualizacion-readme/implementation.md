Las acciones a nivel de código incluyen la lectura de múltiples archivos .md, la creación de nuevos archivos .md en `docs/testing/` y `docs/architecture/`, la edición por bloque del `README.md` raíz, la eliminación por comando de bash `rm` de los archivos md en `src/`, y la ejecución de los comandos `npm run build`, `npm run lint` y `npm run test:all` en `src/`.
