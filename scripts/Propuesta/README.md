# Propuesta (material histórico)

Esta carpeta contiene **documentos de propuesta** y escenarios de uso **anteriores o no vinculados** al estado actual de **GesFer.Product.Front**.

- **No** es la fuente de verdad de configuración, puertos ni variables de entorno.
- La configuración vigente del cliente está en **`src/`** (p. ej. `src/.env.example`, `src/lib/config.ts`).
- El contrato de la API REST lo marca el **backend** vía **OpenAPI** (p. ej. `swagger.json` en el servicio API).

Antes de usar un comando o tabla de estos ficheros, contrastar con la documentación del repo y con el swagger actual del backend.

**Plan de desacoplamiento / orden de capas:** `docs/features/product-front-objetivos-pendientes/plan.md`.
